RegexDelimiter

class RegexDelimiter(jPattern: Pattern) : Delimiter

Regex-based delimiter for complex matching.

Stores the underlying Java Pattern to preserve flags (e.g., CASE_INSENSITIVE, DOTALL) and delegates to Pattern.split, which mirrors JDK behavior (trailing empty segments removed). Immutable and thread-safe; reuse for repeated parsing.
